2. Programming Language Fundamentals

Choose ONE language and master it (Java, C++, or Python)

Java Essentials

Core Concepts

  • OOP principles (Encapsulation, Inheritance, Polymorphism, Abstraction)
  • Classes and Objects
  • Constructors and Destructors
  • Access modifiers (public, private, protected)
  • Static and final keywords

Advanced Topics

  • Interfaces vs Abstract classes
  • Exception handling (try-catch-finally)
  • Collections Framework (ArrayList, HashMap, HashSet)
  • Generics
  • Multithreading basics
  • Lambda expressions and Streams

C++ Essentials

Core Concepts

  • Pointers and References
  • Memory management (new, delete)
  • OOP in C++
  • Constructors and Destructors
  • Operator overloading

Advanced Topics

  • Virtual functions and V-table
  • Shallow vs Deep copy
  • STL (Standard Template Library): vectors, maps, sets
  • Templates and Generic programming
  • Exception handling
  • Smart pointers (unique_ptr, shared_ptr)

Python Essentials

Core Concepts

  • Data types and structures
  • Functions and lambda expressions
  • OOP in Python (classes, inheritance)
  • List comprehensions
  • Decorators basics

Advanced Topics

  • Generators and iterators
  • Exception handling
  • File I/O operations
  • Collections module (Counter, defaultdict)
  • Regular expressions
  • Modules and packages

3. Computer Science Theory

Operating Systems

  • Process Management: Process vs Thread, scheduling algorithms (FCFS, SJF, Round Robin)
  • Memory Management: Paging, segmentation, virtual memory
  • Deadlock: Conditions, prevention, avoidance
  • File Systems: Directory structure, file allocation
  • Synchronization: Mutex, semaphores, critical section

Database Management Systems

  • SQL Basics: SELECT, INSERT, UPDATE, DELETE, JOINs
  • Normalization: 1NF, 2NF, 3NF, BCNF
  • Transactions: ACID properties
  • Indexing: B-trees, B+ trees
  • Keys: Primary, Foreign, Candidate, Composite

Computer Networks

  • OSI Model: 7 layers and their functions
  • TCP/IP: TCP vs UDP, 3-way handshake
  • HTTP/HTTPS: Request/Response, status codes
  • IP Addressing: IPv4, subnetting basics
  • DNS: Domain Name System working

Software Engineering

  • SDLC: Waterfall, Agile, Scrum
  • Design Patterns: Singleton, Factory, Observer (basics)
  • Version Control: Git basics (commit, push, pull, merge)
  • Testing: Unit testing, integration testing
  • SOLID Principles: Basic understanding
